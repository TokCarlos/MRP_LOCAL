# Solucao v0.1.044 - CIMASP REAL_ATA

## Decisao

Aplicar para CIMASP o mesmo criterio usado em SEHIS/GOV RIO: previews reais em PNG no caminho oficial `img/produtos/aco/atas/cimasp/`, mantendo SVGs demo como legado.

## Resultado

- 74 produtos CIMASP existentes atualizados para `imagem.status = REAL_ATA`.
- 16 novos produtos de variacao adicionados com item `N-1`.
- Nomes oficiais preservados sem reescrita.
- Arquivos PNG nomeados com o mesmo radical dos SVGs oficiais atuais.
