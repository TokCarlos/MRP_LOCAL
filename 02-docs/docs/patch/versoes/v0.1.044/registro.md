# Registro v0.1.044 - Catalogacao imagens CIMASP

## Escopo

Catalogacao das imagens reais da ATA CIMASP 029/2025 no catalogo local de produtos.

## Alteracoes

- Migrados 74 previews CIMASP de SVG DEMO para PNG REAL_ATA.
- Criadas 16 variacoes de itens quando o ZIP trouxe imagem adicional do mesmo item.
- Mantidos os nomes oficiais existentes no seed, derivados dos registros/SVGs DEMO atuais.
- Mantidos os SVGs antigos na pasta de imagens como referencia historica/demo.

## Regra aplicada

- O PNG principal substitui o preview do item existente.
- A imagem adicional vira item de variacao com sufixo `-1`, por exemplo `17-1`.
- Nao houve alteracao de HTML, CSS ou JavaScript da tela de produtos.
