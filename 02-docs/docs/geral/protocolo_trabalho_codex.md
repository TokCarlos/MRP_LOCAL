# Protocolo atual de trabalho Codex - MRP_LOCAL

## Leitura obrigatoria

Antes de qualquer trabalho:
1. `02-docs/REGRAS_ATUAIS_MRP.txt`
2. `02-docs/LOG_PROGRESSO_MRP.txt`
3. `COMO_CRIAR_COMANDOS_CODEX.txt`, quando for montar comando.

## Regra de execucao

- Diagnosticar antes de alterar.
- Alterar somente o escopo pedido.
- Documentar resultado.
- Validar com testes cabiveis.
- Nao fazer commit/push sem ordem explicita.

## Area restrita

Alterar `01-mrp` ou `.git` exige chave `LIBERADO`.

## Resposta final esperada

1. Arquivos alterados.
2. O que foi corrigido.
3. O que foi preservado.
4. Validacoes OK.
5. Validacoes pendentes/falhas.
6. Pendencias.
7. Proximo passo recomendado.
