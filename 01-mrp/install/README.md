# install

Ganchos iniciais para instalador/preparador futuro.

Cenario alvo:
- ambiente DEV funciona;
- uma maquina nova nao tem runtime, firewall, tarefa ou atalhos;
- o preparador deve verificar faltas, avisar e orientar instalacao.

Nesta etapa nao baixa instaladores, nao instala dependencias e nao cria backend.
