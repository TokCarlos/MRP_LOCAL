MRP_LOCAL - RESUMO OPERACIONAL

Sistema local-first com frontend estatico em 01-mrp/front_end, porta 8765 e scripts de servico em 03-vs/scripts/servicos.

Esta versao foi reconstruida usando X.zip como base funcional. O frontend e dados operacionais do projeto original foram preservados.

Menu principal: MRP_MENU_SISTEMA.bat
1 - Iniciar Sistema
2 - Desativar Sistema
3 - Reiniciar Sistema
4 - Sair

O start usa caminho resolvido da raiz e inicia o servidor via pushd para funcionar tambem em UNC.
O healthcheck valida index.html em http://127.0.0.1:8765/index.html.
O stop encerra servidores Python http.server ligados a porta configurada.

Ainda pendente de validacao real: tarefa Windows, reboot, queda de rede, queda de energia e watchdog continuo em maquina Windows.
