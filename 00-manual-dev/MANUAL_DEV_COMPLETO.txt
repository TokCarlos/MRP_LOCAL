COMANDOS DE SERVICO

Healthcheck:
powershell -NoProfile -ExecutionPolicy Bypass -File ".\03-vs\scripts\servicos\mrp_frontend_healthcheck.ps1"

EMPACOTAMENTO FUTURO
O MRP_LOCAL deve virar programa instalavel/empacotado. Em maquina limpa, o instalador deve:
1. Executar precheck de Windows/PowerShell/permissoes.
2. Instalar ou fornecer runtime necessario, preferencialmente Python portable.
3. Copiar arquivos para destino configuravel.
4. Gerar configuracao local.
5. Liberar firewall com consentimento.
6. Criar tarefa/watchdog.
7. Executar healthcheck.
8. Entregar relatorio de instalacao.

OBSERVACAO
O instalador prepara ambiente. Regra de negocio continua no motor/backend e nao pode depender do instalador ou do caminho fisico.
